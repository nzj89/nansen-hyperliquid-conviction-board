# Nansen Hyperliquid Conviction Board

Public static demo of a trader-facing dashboard built from the Hyperliquid + Nansen CLI concept work.

What this demo shows:

- distilled long ideas instead of raw wallet spam
- conviction clusters from top traders
- early / workable / late timing reads
- hold-window estimates and exit pressure
- wallet-level drill-down with copyable addresses

This public site is a demo build using the generated static snapshot in `data/`.
